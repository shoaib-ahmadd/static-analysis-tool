# 🚀 Static Code Analysis Tool

A lightweight static code analysis tool that helps detect common issues in Python code without executing it. This project improves code quality by identifying bugs, bad practices, and code smells.

---

## 📌 Features

- 🔍 Detects common code issues  
- 🐞 Identifies potential bugs  
- ⚡ Fast static analysis (no execution required)  
- 📊 Generates code quality score  
- 🧠 Beginner-friendly implementation  

---

## 🛠️ Tech Stack

- Backend: Python (Flask)  
- Frontend: HTML, CSS  
- Core Logic: Custom static analyzer  

---

## ⚙️ How It Works

1. Upload a Python (.py) file  
2. The analyzer processes the code  
3. Detects:
   - Missing docstrings  
   - Unused imports  
   - Long lines  
   - Formatting issues  
4. Generates a score and displays issues
